# WorkHomeHtmlCss

Mon premier projet web ! 🎓

## 📋 Description

Ce référentiel regroupe l'ensemble de mes premiers pas en HTML et CSS. Il contient tous mes devoirs, exercices pratiques et projets réalisés durant ma formation à l'université.

C'est mon espace d'apprentissage où je mets en pratique les concepts fondamentaux du développement web front-end(Html/Css-Javascripts).

## 🎯 Objectifs d'apprentissage

- Maîtriser les bases de HTML5
- Apprendre les fondamentaux de CSS3
- Pratiquer la mise en page et le design responsive
- Améliorer progressivement mes compétences

## 🛠️ Technologies utilisées

- HTML5
- CSS3


## 🚀 Comment utiliser

1. Clonez le référentiel :
   ```bash
   git clone https://github.com/lenoirgn/WorkHomeHtmlCss.git
   cd WorkHomeHtmlCss
   ```

2. Ouvrez n'importe quel fichier `index.html` dans votre navigateur web
