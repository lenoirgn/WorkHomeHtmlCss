Elément <h2>

Un paragraphe avec un peu de contenu pour illustrer l'utilisation d'une feuille CSS externe. Etudiez le code source de ce document pour trouver la déclaration de l'utilisation de la feuille de style style1.css.

Voici le code contenu dans la feuille de style style1.css utilisée par cette page : 

/* définition de style pour l'élément <body> */
body {
	/* on attribue la valeur "rgb(248,248,248)" à la propriété "background-color" (= couleur d'arrière-plan) pour l'élément <body> */
	background-color	: rgb(248,248,248); /* valeur de couleur en rgb */
}

h1 {
	color			: yellow;		/* valeur de couleur nommée  = #FFFF00 = rgb(255,255,0)*/
	background-color	: navy;
	font-weight		: bold;
	font-size		: 24px;
}

p {
	background-color	: #888888;		/* valeur de couleur en hexadécimal */
	color			: #FFFFFF;
	font-size		: 14px;
}

pre {
	color 			: navy;
    background-color :white;
}

/* un élément <strong> n'est pas forcément "en gras", ici il sera "rouge et en italique" */
strong {
	color 			: yellow; 
	font-style		: italic;
	font-weight		: normal;
}

abbr {
	text-decoration		: underline;
}

Le style s'applique à tous les éléments de même type.
