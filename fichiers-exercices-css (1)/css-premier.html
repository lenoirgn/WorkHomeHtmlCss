La balise <pre>

Nous allons essayé de découvrir le rôle de la balise <pre>. 

D'après vous quel est le rôle de cette balise <pre> ?

Faites-vous une idée en regardant comment s'affiche les lignes suivantes :

def max(x,y):
   if x > y:
      return x
   else:
      return y
	  
Vous pouvez par exemple comparer avec ce que donnerait l'affichage du contenu de cet élément <pre> s'il était placé dans un élément <p>...

Le style s'applique à tous les éléments de même type.
